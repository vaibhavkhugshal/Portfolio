# Personal Portfolio Website

A modern and fully responsive personal portfolio website built using HTML, CSS, and JavaScript.

This portfolio showcases my:
- Skills
- Education
- Contact Information

Designed with a clean UI, responsive layouts, glassmorphism effects, and smooth user experience across all devices.

## Features

- Fully Responsive Design
- Modern Glassmorphism UI
- Smooth Scrolling Navigation
- Responsive Hero Section
- Skills Grid Layout
- Education Timeline/Cards
- Contact Form Integration
- Optimized for Mobile Devices
- Clean and Minimal Design

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- Responsive Design Principles

## Responsiveness

The website is optimized for:
- Mobile Phones
- Tablets
- Desktops
